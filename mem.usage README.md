
To run this script


# Create a new project

* Open CMD
* Type `cd c:\repos\`
* Type `mkdir my_new_folder`
* Type `cd my_new_folder`
* Type `virtualenv myvenv`

* Open CMD
* Type `cd c:\repos\monitor`
* Type `myvenv\scripts\activate`
* Type `python disk_monitor.py`
